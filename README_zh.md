# Hermes V2 微内核 (Microkernel)

Hermes V2 微内核是为 Hermes Agent 设计的轻量级、安全且极具韧性的基础架构。它提供强大的环境防护、全局自愈能力以及优化的配置管理，确保智能体运行环境的绝对稳定。

## 核心特性

- **安全的 MCP 桥接**: 为 MCP 工具提供安全隔离的桥接脚本，通过严格的环境变量解析防止任意代码执行和凭证泄露。
- **防污染 Shell 拦截器**: 在 Shell 配置文件（`.bashrc`、`.zshrc`）中注入防护层，支持多行深度检测，防止 YAML 解析逃逸导致的配置固化。
- **全局自愈插件**: 自动修复损坏的配置文件，并具备严格的类型检查机制，防止系统崩溃。
- **微内核技能组**: 内置必要的子智能体协同和系统维护技能，实现无缝的自治操作。
- **跨平台兼容**: 使用 POSIX 标准规范，全面兼容 Linux、macOS 和 BSD 操作系统。

## 安装指南

运行安装脚本以部署微内核架构：

```bash
chmod +x install.sh
./install.sh
```

## 目录结构
- `install.sh`: 核心自动化部署脚本。
- `plugins/global_auto_heal/`: 提供配置韧性的全局自愈插件。
